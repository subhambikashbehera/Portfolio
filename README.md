# Subham Bikash Portfolio

Static portfolio site configured for GitHub Pages.

## Publish for free with GitHub Pages

1. Create a GitHub repository, for example `Portfolio`.
2. Push this folder to the repository's `main` branch.
3. In GitHub, open the repository settings and enable Pages from `main` / root.
4. Keep the custom domain as `subhambikash.com`; this repository already includes the required `CNAME` file.
5. In GoDaddy DNS, point the root domain and `www` host to GitHub Pages.

Recommended DNS records:

```text
Type   Name   Value
A      @      185.199.108.153
A      @      185.199.109.153
A      @      185.199.110.153
A      @      185.199.111.153
CNAME  www    subhambikashbehera.github.io
```

If your GitHub username is different, replace `subhambikashbehera.github.io` with `<your-github-username>.github.io`.
